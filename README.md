mongo-cheatsheet
================

A cheatsheet for [MongoDB](http://mongodb.org/).

Uses wkhtmltopdf and Makefile to render the PDF.

Currently contains:

* Querying with JSON.
* Datatype table.

Building
--------

    make doc-pdf
