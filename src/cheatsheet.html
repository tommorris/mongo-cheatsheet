MongoDB Cheatsheet

Operators

$lt
less than
$lte
less than or equal to
$gt
greater than
$gte
greater than or equal to
$ne
not equal to
$in
is in array
$nin
not in array
$exists
key exists - takes boolean true or false
$mod
fast modulo - where a % 10 == 1, provide array of [10, 1]
$all
array must contain all values specified in argument array
$size
array size
$type
matches on type - see type table below
$or
Boolean or expression to do queries.
e.g. {$or : [ {a : 1 }, {b : 2} ] }
$not
Negate check - {x: {$not : {$exists : true }}}
$where
JavaScript expression that evaluates to true or false.
e.g. { $where : "this. a > 3" }
